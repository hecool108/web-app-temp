# Web App Temp
bunx
```bash
bunx degit https://github.com/hecool108/web-app-temp  your-app-name
cd your-app-name
```

To install dependencies:

```bash
bun install
```

To Dev:

```bash
bun dev
```

This project was created using `bun init` in bun v1.1.18. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
